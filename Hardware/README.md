# 器件选型：
    主控芯片：    STM32F103RCT6 
    电机驱动：    TB6612FNG 
    角度传感器:   MPU6050 
    USB转UART：  CH340 
    Buck芯片(12V转5V)   MP2359 
    Buck芯片(12V转11V)  LM2596-ADJ 
    
# 预留接口：
    OLED12864 SPI 接口 
    HC-04 蓝牙模块接口 
    NRF2401 2.4GHZ模块接口 
    预留了三路电位器，可以使用其调节参数 
    2个用户自定义按键 
    空闲I/O引出
